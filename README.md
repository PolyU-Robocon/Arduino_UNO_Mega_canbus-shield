# Arduino_UNO_Mega_canbus-shield
The USB can bus shield for arduino

# V2.1 release

## feature 

* SSD 1306 (i2C) support
* 1 x user led 
* 2x user button
* MPU9260 connector
* 24v DC input with step down
* MCP 2515 -> TJA1050 can bus support

## file support
KiCad 6.0 or above
